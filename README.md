# AdventOfCode2024
Advent of code [2024](https://adventofcode.com/2024)

All code written in C++. 

## How to build

 - cd into a days directory i.e. cd Day_1
 - mkdir build
 - cd build
 - cmake ..
 - cmake --build .

## How to run

After build is complete, from inside the build directory
 - cd bin
 - ./{dayName} i.e. ./day_one path/to/input/file.txt
